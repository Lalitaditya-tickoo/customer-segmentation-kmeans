\# Customer Segmentation using K-Means



An interactive Machine Learning project that segments customers into different groups using K-Means clustering.



\---



\## 🚀 Features



\- K-Means clustering

\- Elbow method

\- PCA visualization

\- Streamlit UI

\- Real-time prediction
